## Brave
<img src="https://raw.githubusercontent.com/wiki/OpenUserJS/OpenUserJS.org/images/brave_icon.min.svg?sanitize=true" width="128" height="128" align="right">

Brave is a free web browser based off of [Chromium][chromium]. Brave's selling points are more security, better privacy, speed, simplicity and stability. The desktop version runs on Windows, Linux and OS X, and there are mobile versions for Android and iOS.

To run userscripts on Brave, you normally need a manager extension.
